# Contributing

Thanks for your interest in Giit.

Giit is currently in private development. This public repository is intended for
project information, community feedback, and selected issue tracking rather than
general source contributions.

## Helpful Contributions

At this stage, the most useful public contributions are:

- Clear bug reports.
- Safety and recovery workflow feedback.
- Feature requests grounded in real Git workflows.
- UX feedback from developers who use Git heavily.
- Documentation corrections or clarification requests.

## Pull Requests

Code pull requests are not currently expected unless a maintainer has asked for
one. Small documentation fixes may be accepted if they match the current public
scope of the project.

## Good Issues

When opening an issue, please include:

- What you were trying to do.
- What happened.
- What you expected to happen.
- Your operating system.
- Any relevant Git repository state, with private details removed.
- Screenshots or short screen recordings when they clarify the problem.

For security concerns, do not open a detailed public issue. See
[SECURITY.md](SECURITY.md).
